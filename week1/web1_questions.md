- 브라우저의 주요 기능이 무엇일까요?
- 과거와 달리 최근에 대부분의 브라우저가 표준 HTML, CSS 명세를 따르는 이유는?
- 브라우저의 UI는 표준 명세가 없음에도 서로 닮은 이유가 무엇일까요?
